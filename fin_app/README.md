# Financial Management Application
Financial Management Application is designed to help users effectively manage their finances by tracking income, expenses, and creating budgets. With intuitive features such as categorizing transactions and generating customized reports, users can get a clear overview of their financial situation and make informed decisions.


## Technology 
- Front-end: Flutter 
- Database: Firebase
- Back-end: Hive
## Demo
### Run project
- Use `git clone` to clone this project 
- Use `flutter pub get` in terminal to get dependencies
- Run project by `flutter run`
